BASIC COMMANDS : 

pwd - present working directory ( shows ur current file/folder path from home)
ls - list out all files nd folders of ur terminal
cd- change directory (cd foldername) or go back into main working directory 
mkdir - make directory (folder)
rmdir - remove directory
rm - files or folders 
cp -r(for folders) || cp for files - copy files and folders 
mv - move or rename files 
touch - create files 
find - search for files nd directories 
locate - find a file by name 
stat show detailed info about file 
tree - display files in a directory as tree structure

FILE PERMISSION AND OWNERSHIP COMMANDS:

chmod - change file permissions
chown - change file ownership

PROCESS MANAGEMENT 

ps : display current running processes
top: show real time processes
htop: interactive process viewer
kill - terminate a process by PID
pkill - kill a process by name
nice - start a process with specific priority
jobs : display bg jobs
bg : resume a bg job 
fg : resume a job in the foreground.

ss - show active socket connections
wget : download files from web 
curl : transfer data from url 
df : show diskspace usage 
du : show file space usage
uptime : show system uptime
